## 📖 문자열 (String)

`String` -> 
 - `int` - Integer.parseInt()
 - `Double` -  Double.parseDouble()
 - `Long` -  Double.parseDouble()
 - `char` - *str*.charAt()

<br>

## 📖 숫자 (int)

`int` ->
 - String - Integer.toString()
 - Long - Long.valueOf()
 - Double - Double.valueOf()

<br>

## 📖 실수 (double)

`double` ->
 - int - Double.valueOf()

<br>
