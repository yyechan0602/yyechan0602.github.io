## 📘 JSON vs XML

### 📌 JSON

 - 태그로 표현하는 대신, 프로그래밍 언어의 리스트처럼 `"`와 `,`를 사용하여 데이터를 나타낸다.

### 📌 XML

 - 기본적으로 `HTML`을 기반으로 만들어져서 태그 시스템이 존재한다.
