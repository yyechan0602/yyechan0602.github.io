## 📖 최소 신장 트리(MST)

### 🍄 신장 트리(Spanning Tree)  

 - `연결 그래프`에서 그래프 탐색을 통해서 생기는 트리 엣지로 이루어진 `사이클을 갖지 않는 최소 부분 그래프`를 의미한다.
 - 그래프의 모든 정점이 서로 연결되어 있으면서, 순환구조가 반복되지 않는다.
 - 최소 부분이란 - 모든 정점을 포함하고, `간선의 개수 : 정점의 개수 -1`
