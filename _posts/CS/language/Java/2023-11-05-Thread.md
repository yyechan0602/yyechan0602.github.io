## 📖 쓰레드

인터페이스란 자바 프로그래밍 언어에서 클래스들이 구현해야 하는 동작을 지정하는데 사용하는 추상 자료형으로 일종의 설계도와 유사하다.  
무조건 구현을 통하여 실사용이 가능하며, 인터페이스 자체에는 기능 명시 이외에 어떠한 기능도 없다.
