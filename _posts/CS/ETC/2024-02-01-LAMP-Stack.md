## 📖 LAMP 스택

LAMP은 웹사이트나 서버 운영에 자주 쓰이는 이름들의 약자이다

 - Linux
 - Apache
 - MySQL
 - PHP

프로그램 본체 + 프로그램 런타임 + 웹 서버 + DB로 운영되는 Linux 기반의 프로그램을 의미한다.
